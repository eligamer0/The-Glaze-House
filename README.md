# The Glaze House 🍩

Sitio web para mostrar tipos de donas y postres.

## 🧪 Tecnologías usadas
- React Vite
- React Router DOM
- CSS
- JavaScript

## 🎯 Funcionalidades
- Navegación por rutas reales
- Vista de productos
- Sección de contacto que lleva al footer
